# Transfer Acceleration Overview

## What is Transfer Acceleration?

S3 Transfer Acceleration speeds up uploads and downloads between clients and S3 buckets using AWS's global edge network.

## Problem It Solves

Users located far from the S3 bucket region often experience higher latency and slower transfer speeds.

## Solution

Traffic enters the nearest AWS Edge Location and travels across AWS's optimized backbone network.

## Benefits

- Faster uploads
- Faster downloads
- Better global user experience
