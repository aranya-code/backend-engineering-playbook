# Storage Class Overview

## Why Storage Classes Exist

Different data has different access patterns.

Examples:
- Frequently accessed images
- Monthly reports
- Yearly archives

Using a single storage tier for all data would be expensive.

AWS provides multiple storage classes optimized for:
- Cost
- Availability
- Retrieval Speed

## Exam Rule

The less frequently data is accessed:
The cheaper storage becomes.
The slower retrieval may become.
