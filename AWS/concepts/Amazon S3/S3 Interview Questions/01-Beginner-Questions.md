# Beginner Interview Questions

Q1. What is Amazon S3?
A. Amazon S3 is AWS's scalable object storage service used to store and retrieve data.

Q2. What are Buckets?
A. Buckets are logical containers used to store S3 objects.

Q3. What is an Object?
A. An object consists of data, metadata, tags and a unique key.

Q4. What type of storage is S3?
A. Object Storage.

Q5. Maximum object size?
A. 5 TB.
