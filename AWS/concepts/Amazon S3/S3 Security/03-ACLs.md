# Access Control Lists (ACLs)

## Definition
Legacy access control mechanism for buckets and objects.

## Types
- Bucket ACL
- Object ACL

## Common Permissions
- Read
- Write
- Read ACP
- Write ACP
- Full Control

## AWS Recommendation
Use IAM and Bucket Policies instead of ACLs whenever possible.

## Exam Tip
ACLs are considered legacy.
