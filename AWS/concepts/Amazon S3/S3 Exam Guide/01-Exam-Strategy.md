# SAA-C03 Exam Strategy

## How AWS Tests S3

AWS rarely asks direct definition questions.

Instead, questions focus on:
- Business requirements
- Cost optimization
- Availability
- Security
- Compliance

## Exam Approach

Identify:
1. Security Requirement
2. Cost Requirement
3. Availability Requirement
4. Performance Requirement

Then choose the S3 feature that satisfies all requirements.
