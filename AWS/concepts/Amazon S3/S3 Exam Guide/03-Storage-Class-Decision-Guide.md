# Storage Class Decision Guide

Need frequent access?
-> S3 Standard

Need infrequent access?
-> Standard-IA

Need archive with immediate access?
-> Glacier Instant Retrieval

Need cheapest archival storage?
-> Glacier Deep Archive

Unknown access pattern?
-> Intelligent-Tiering

Exam Tip:
Look for keywords:
'rarely accessed'
'archive'
'long-term retention'
