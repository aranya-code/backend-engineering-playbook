# Amazon MemoryDB for Redis

## Definition

Redis-compatible durable in-memory database.

---

## Features

- Redis compatible
- Durable storage
- Multi-AZ transactional log
- Ultra-fast performance

---

## Use Cases

- Gaming
- Real-time apps
- Financial systems
- Session storage

---

## Difference from ElastiCache

MemoryDB focuses more on durability than caching.
