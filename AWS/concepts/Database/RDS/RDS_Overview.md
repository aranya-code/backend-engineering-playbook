# Amazon RDS Overview

## What is Amazon RDS?

Amazon RDS is a fully managed relational database service provided by AWS.

### Benefits
- Automated backups
- OS patching
- Monitoring
- Scaling
- High availability
- Disaster recovery

### Supported Engines
- MySQL
- PostgreSQL
- MariaDB
- Oracle
- SQL Server
- Aurora

---

## Advantages Over Running DB on EC2

- AWS handles maintenance
- Point-in-Time Recovery
- Multi-AZ support
- Read replicas
- Monitoring dashboards
- Easier scaling

---

## Important Note

You cannot SSH into standard RDS instances.

Only RDS Custom supports OS-level access.

---

## Interview Notes

RDS reduces operational overhead significantly compared to self-managed databases.
