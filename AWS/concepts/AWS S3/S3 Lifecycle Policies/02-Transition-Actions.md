# Transition Actions

## Definition

Transition actions move objects between storage classes.

## Common Example

30 Days  -> Standard-IA
90 Days  -> Glacier Flexible Retrieval
365 Days -> Deep Archive

## Supported Targets

- Standard-IA
- One Zone-IA
- Intelligent-Tiering
- Glacier Instant Retrieval
- Glacier Flexible Retrieval
- Glacier Deep Archive

## Why Transition?

Older data is usually accessed less frequently.
