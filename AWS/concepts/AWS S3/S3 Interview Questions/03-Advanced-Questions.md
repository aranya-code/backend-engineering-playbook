# Advanced Interview Questions

Q1. Explain Object Lock.
A. Prevents modification or deletion of protected objects.

Q2. Difference between Governance and Compliance Mode?
A.
Governance = Authorized bypass possible.
Compliance = No bypass possible.

Q3. Explain Multi-Region Access Points.
A. Provides a global endpoint that routes requests to optimal regional buckets.

Q4. Explain Storage Lens.
A. Analytics service providing storage visibility and optimization insights.
