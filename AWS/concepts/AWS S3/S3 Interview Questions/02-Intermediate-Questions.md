# Intermediate Interview Questions

Q1. What is Versioning?
A. A bucket-level feature that maintains multiple versions of an object.

Q2. Difference between CRR and SRR?
A.
CRR = Cross Region Replication
SRR = Same Region Replication

Q3. What is a Presigned URL?
A. Temporary URL providing time-limited access to a private object.

Q4. What is Intelligent Tiering?
A. Automatically moves objects between storage tiers based on access patterns.
