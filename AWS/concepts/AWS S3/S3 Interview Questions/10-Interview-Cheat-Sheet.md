# Interview Cheat Sheet

Must Remember:

- S3 = Object Storage
- Bucket Names = Global
- Max Object Size = 5 TB
- Versioning protects deletes
- CRR = Disaster Recovery
- SRR = Same Region
- SSE-KMS = Enterprise Encryption
- Deep Archive = Lowest Cost
- Presigned URL = Temporary Access
- Event Notifications -> Lambda/SNS/SQS
