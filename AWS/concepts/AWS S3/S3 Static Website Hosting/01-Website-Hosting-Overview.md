# Static Website Hosting Overview

## What is Static Website Hosting?

Amazon S3 can host static websites directly from a bucket.

Supported:
- HTML
- CSS
- JavaScript
- Images

Not Supported:
- Server-side code
- Databases
- Backend processing

## Common Use Cases

- Portfolio Websites
- Landing Pages
- Documentation Sites
- React/Vue/Angular Builds

## Benefits

- Low Cost
- High Availability
- Easy Deployment
