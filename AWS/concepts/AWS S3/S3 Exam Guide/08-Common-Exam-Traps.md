# Common Exam Traps

Trap:
Choose Glacier when immediate access is required.

Correct:
Glacier Instant Retrieval

Trap:
Use public buckets unnecessarily.

Correct:
Presigned URLs

Trap:
Forget Versioning requirement for Replication.

Correct:
Enable Versioning on both buckets.

Trap:
Confuse durability and availability.

Remember:
Durability = Data survives
Availability = Data accessible
