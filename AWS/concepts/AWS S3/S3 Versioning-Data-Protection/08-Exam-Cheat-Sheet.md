# Exam Cheat Sheet

- Versioning protects against accidental deletes.
- Versioning protects against overwrites.
- Delete Marker is not actual deletion.
- MFA Delete requires Versioning.
- Object Lock supports WORM.
- Compliance Mode cannot be bypassed.
- Governance Mode can be bypassed by authorized users.
- Best practice: Enable Versioning for production workloads.
