# AWS CLI

## Version

```
### Gives the version
aws --version
```

## Configure
```
### AWS CLI configuration
aws configure
```

### You will be prompted for:

```text
AWS Access Key ID:
AWS Secret Access Key:
Default region name:
Default output format:
```

### Example:

```text
AWS Access Key ID: AKIAXXXXXXXX
AWS Secret Access Key: ****************
Default region name: ap-south-1
Default output format: json
```

## List IAM Users

```bash
aws iam list-users
```
