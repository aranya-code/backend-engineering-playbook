# 📖 About

This repository is my personal Kubernetes knowledge base built while learning Kubernetes from the ground up.

Instead of keeping scattered notes, I decided to create a structured playbook that covers:

- Kubernetes fundamentals
- Core architecture
- Kubernetes resources
- Networking
- Storage
- Helm
- Minikube
- kubectl CLI
- Interview preparation
- Best practices

The goal is to make this repository useful for:

- Students
- Backend Developers
- DevOps Engineers
- Software Engineers
- Anyone preparing for Kubernetes interviews
